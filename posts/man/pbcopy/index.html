Name

pbcopy - provide copying and pasting to the clipboard from command line

Synopsis

pbcopy

Description

pbcopy takes the standard input and places it in the clipboard (there are different boards available; cf. man pbcopy for more details).

Example

pbcopy < $HOME/.ssh/id_ed25519.pub
